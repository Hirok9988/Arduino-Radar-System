# Arduino Radar System

This project demonstrates a mini radar system using Arduino, ultrasonic sensor, and servo motor with real-time visualization using Processing.

## Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Buzzer
- Breadboard and Jumper Wires

## How It Works
- The servo motor rotates the ultrasonic sensor across angles
- Distance is measured using ultrasonic sensor
- Data is sent to Processing via serial communication
- Processing displays radar-style visualization
- Buzzer alerts when object is detected within range

## Folder Structure
- arduino_code/ → Arduino source code
- processing_code/ → Processing visualization code
- connections/ → Wiring diagram
